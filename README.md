# 📚 AI Based Paper Prediction & Learning System – Frontend

This is the frontend application for the **AI Based Paper Prediction and Learning System**. It provides students with a modern, fast, and interactive interface to generate practice papers, attempt quizzes, and track their exam preparation progress.

Built with **Next.js**, **Tailwind CSS**, and **TypeScript**, this frontend communicates with a Flask-based backend API and an AI model to deliver personalized learning experiences.

---

## 🎯 Features

- **User Authentication** – Sign up, login, and profile management.
- **Class & Board Selection** – Choose between Punjab Board (9th–12th) or Cambridge (O/A Levels).
- **Generate Full Papers** – Request AI-generated predicted papers based on past exam patterns.
- **Generate Quizzes** – Get short, topic-wise quizzes for quick revision.
- **Timed Test Environment** – Attempt generated papers with a built-in timer.
- **Auto-Grading & Results** – View scores, correct/incorrect answers, and topic-wise feedback.
- **Performance Dashboard** – Track past attempts and monitor improvement over time.
- **Fully Responsive** – Works seamlessly on desktops, tablets, and mobile devices.

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ and npm / yarn / pnpm installed
- Backend server running (Flask API) – [link to backend README]

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ArslanGujjar123/NextQ-frontend.git
   ```
2. cd frontend

## Run

3. npm run build
4. npm start
